When the Expansion Market is installed, the mod does not know what map you are using, so it does not install actual Traders or a Physical Marketplace. Let that sink in. It also does not
add the Expansion types/spawnabletypes/events files to the server. You must do that.

This is everyone's first panic moment, when they teleport proudly to Green Mountain and find that there's nothing there. These folders have the Traders & the physical Marketplace 
.map files that create the Market where players can go and do their thing. They are in map specific folders, so pick your map and copypasta.

1. Append your existing cfgeconomycore.xml file with the commented section in the Repository's cfgeconomycore.xml file. Only add what is commented to your existing file.

Don't know what a cfgeconomycore.xml file is or why it's important? https://youtu.be/bvf4t9zJW7s?si=Dl2juBXLrriyco_C

2. Append the commented section in the Repository's cfgeventspawns.xml file to your cfgeventspawns.xml file. Only add what is commented to your existing file.

3. Copy and paste the objects & traders folder into your missions/expansion folder. This will add the .map files to create the Traders & Market locations to your server.

Note: The Chernarus Chernogorsk_Ship.map & Gorka_HuntingHouse.map are just built up (cool) locations you can mess with once you know what you're doing. They're empty.

That's it. Teleport to Green Mountain, Kamenka or any of the other locations and you have NPCs there to sell stuff. Enjoy!
